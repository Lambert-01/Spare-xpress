# Spare Xpress

A web-based spare parts management system for automotive spare parts.

## Features

- Product management
- Order processing
- Invoice generation
- User authentication
- Email notifications

## Installation

1. Clone the repository
2. Run `composer install` to install PHP dependencies
3. Configure database in `includes/config.php`
4. Access via web server

## Usage

Navigate to the main page and use the admin panel for management.
